PokeJump

Instructions

- The Pikachu is controlled using the following Hand Gestures.

  • Palm: Makes the Pikachu jump.
  • Two Fingers: Pauses the pokemon in Jump Mode.

- The objective is to survive from the other Pokemon.

- Collision results in reduction of health by 10%.

- If the Pikachu stays in 'Pause Mode' for 6 seconds, the game gets ended.

- The game ends when the timer runs out in pause mode, or all the health becomes zero.

Documentation

This hand gesture-controlled Pokemon game is created using Teachable Machine

The general machanics of the game were created using simple p5.js techniques, initially the pikachu being given keyboard controls. Then, a model was trained on Teachable machine to detect 'Palm', 'Two Fingers', and an IDLE state. The keyboard controls were then replaced, with a few other changes to make the program compatible with hand gestures instead of the keyboard controls.
